# Phone Number and Website Checker

## Overview
Phone Number and Website Checker is a web-based application developed using HTML, CSS, PHP, and MySQL. The system allows users to register, log in, and validate phone numbers and website URLs.

## Features
- User Registration
- User Login and Logout
- Phone Number Validation
- Website URL Validation
- Responsive User Interface
- Database Connectivity using MySQL

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP

## Project Files

- index.php – Home page
- register.html – Registration form
- register.php – Registration processing
- login.html – Login form
- login.php – Login processing
- logout.php – User logout
- check_phone.php – Phone number validation
- check_website.php – Website URL validation
- create_database.php – Database creation script
- style.css – Application styling

## Installation

1. Install XAMPP.
2. Start Apache and MySQL.
3. Place the project folder inside `htdocs`.
4. Import or create the database using `create_database.php`.
5. Open the application:

## How to Use

1. Register a new account.
2. Login using your credentials.
3. Choose Phone Number Checker or Website Checker.
4. Enter the required input.
5. View the validation result.

## Project Objective

The objective of this project is to validate phone numbers and website URLs using PHP validation techniques while providing secure user authentication and database support.
